Landing Pages - HTML & CSS Practice
Este proyecto consiste en 55 páginas enlazadas entre sí, desarrolladas únicamente con HTML y CSS.

📖 Descripción
El objetivo principal de este proyecto es practicar el diseño y la estructura de interfaces web sin utilizar JavaScript ni funcionalidades interactivas.

Cada página funciona como una pequeña interfaz visual, enfocada en:

Diseño visual
Organización de contenido
Uso de estilos con CSS
Estructuración semántica en HTML
Enlace entre páginas (navegación básica)

No incluye interacción dinámica, ya que el enfoque es mejorar habilidades de maquetación y diseño.

🎯 Objetivo del proyecto
Mejorar habilidades en diseño web
Practicar estructura HTML
Aplicar estilos modernos con CSS
Crear múltiples interfaces visuales
Entender la navegación entre páginas

🛠 Tecnologías utilizadas
HTML
CSS3

📂 Estructura
El proyecto contiene 55 páginas enlazadas entre sí, simulando diferentes diseños e interfaces visuales.
